# LINE WORKS Talk Exporter

LINE WORKSのトーク画面からメッセージを取得し、テキストファイルとして出力するChrome拡張機能です。

## 機能

- LINE WORKSのトーク画面からメッセージを取得
- 話者名と発言時刻を含む読みやすい形式で出力
- `.txt`ファイルとしてダウンロード

## 出力フォーマット

```
LINE WORKS トーク履歴
出力日時: 2024/5/20 12:00:00
==================================================

田中 太郎 (10:00):
「おはようございます。」

自分 (10:01):
「おはようございます。本日の会議の件ですが…」
```

## インストール方法

### 1. リポジトリをクローン

```bash
git clone https://github.com/YOUR_USERNAME/LINE-WORKS-トーク取得.git
cd LINE-WORKS-トーク取得
```

### 2. Chrome拡張機能として読み込み

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions` と入力して移動
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
5. このプロジェクトのフォルダを選択

## 使用方法

1. LINE WORKSのトーク画面をブラウザで開く
2. 画面右上の拡張機能アイコンから「LINE WORKS Talk Exporter」をクリック
3. 「トークを保存 (.txt)」ボタンをクリック
4. メッセージが自動的にテキストファイルとしてダウンロードされます

## 技術仕様

- **Manifest Version**: 3
- **対応ブラウザ**: Google Chrome
- **権限**: `activeTab`, `scripting`

## ファイル構成

```
.
├── manifest.json      # 拡張機能の設定ファイル
├── popup.html        # ポップアップUIのHTML
├── popup.js          # ポップアップの動作ロジック
├── content.js        # メッセージ抽出ロジック
├── styles.css        # UIスタイル
└── README.md         # このファイル
```

## 注意事項

- LINE WORKSの画面構造が変更された場合、正しく動作しない可能性があります
- メッセージが取得できない場合は、ページをリロードしてから再試行してください
- 取得できるのは画面に表示されているメッセージのみです（スクロールして読み込まれていない過去のメッセージは取得できません）
